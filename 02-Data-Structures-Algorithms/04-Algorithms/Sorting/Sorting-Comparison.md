# Sorting Comparison

Overview

Compare common sorting algorithms by complexity and characteristics.

Table (summary):
- Bubble Sort: O(n^2), stable, simple
- QuickSort: Average O(n log n), unstable, fast in practice
- MergeSort: O(n log n), stable, needs extra space
- HeapSort: O(n log n), in-place, not stable

When to use
- Use MergeSort for stability and predictable performance.
- QuickSort for in-memory general-purpose sorting with tuned pivot.
